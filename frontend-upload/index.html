How it works

This application uses Assembly AI's powerful Speech-to-Text API directly from your browser - no backend server needed! Files are uploaded directly to Assembly AI, transcribed, and the results are sent to your n8n webhook.

Simply drag and drop your file or click to browse. Assembly AI automatically handles all audio and video formats, extracts audio from video files, and provides high-quality transcriptions with punctuation and formatting.

Key Features

  • Large File Support: Up to 512MB file size (full Assembly AI capacity)
  • No Server Needed: Direct browser-to-Assembly AI integration
  • Auto Format Detection: No need to specify audio format
  • Video Support: Automatically extracts audio from video files
  • High Accuracy: Industry-leading transcription quality
  • Fast Processing: Direct API calls, no server bottlenecks

Supported Formats

Audio: MP3, WAV, FLAC, M4A, AAC, OGG, OPUS, and more

Video: MP4, MOV, AVI, MKV, WebM, FLV, 3GP, and more

Assembly AI automatically processes any audio or video format and extracts the best possible transcription.
